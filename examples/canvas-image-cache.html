Image Cache with Canvas

Demonstrates the use of the cacheExternalGraphic style to avoid flickering on the Canvas Renderer. The Left Image is always reloaded on redraw while the Right Image is cached. After redrawing the map (after pan for example), the Left Image will always be above the right one.

Click on the features above to see them selected. This example uses the Canvas renderer so it only works on browsers where canvas is supported.

View the canvas-hit-detection.js source to see how this is done.
