Google (v3) Layer Example With Custom Styled Map Example

Google, api key, apikey, light

Demonstrate use the Google Maps v3 API to apply styling on Map.

Google is encouraging the use of keys for Google Maps APIs as described here: https://developers.google.com/maps/documentation/javascript/get-api-key. Once you have included the Google Maps script in your html, refer to the google-map-set-style.js source to see how to use Google Maps as layer with custom style in OpenLayers.
